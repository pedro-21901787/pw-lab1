HTML5 e CSS

HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.


Elementos do HTML5
header done indica cabeçalho.
nav done indica menu de navegação com links para as páginas.
main done indica conteúdo principal e único, que não se repete.
article clear indica parte do documento que pode ser vista de forma independente.
section clear especifica uma parte do conteúdo, usada em conjunto com etiquetas h.
aside clear indica anotações, publicidade ou outros assuntos desconexos.
footer done indica rodapé da secção.
details clear define detalhes adicionais que o utilizador pode ver ou esconder, num elemento summary.
meter clear define uma medida escalar estática dentro de um intervalo.
progress clear define uma barra de progresso de uma tarefa.
abbr clear indica o significado de uma abreviatura em conjunto com o atributo title.
address clear informação sobre o contacto do autor.
